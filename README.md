# Tetris in C++

Hi there is a Free time project driven by curiosity from me.😅

This a classic Tetris clone developed from scratch using C++. This project serves as a hands-on exploration of core game development concepts, including game loops, state management, and collision detection.

## Project Structure
This repository is specifically configured to track the standalone game environment:
* `/Tetris GameBoy Proj/` - Contains all the C++ source code, headers, and necessary assets for the game.

## Roadmap & Future Development
This project is an ongoing exploration of game development. Future iterations and planned features include:
* **Graphical Interface:** Transitioning from console-based rendering to a full 2D graphical engine (using libraries like SFML or SDL2) for improved visuals, animations, and UI.
* **Game Boy Port:** Developing a dedicated retro version of this Tetris clone playable on classic Nintendo Game Boy hardware, utilizing tools like GBDK (Game Boy Development Kit) to compile C code into playable ROMs.

## Prerequisites
To compile and run this game, you will need a C++ compiler installed on your system (such as GCC/G++ for Linux/Windows or Clang for macOS).
